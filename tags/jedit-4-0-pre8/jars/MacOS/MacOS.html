Mac OS Plugin Version 1.1 (Feb. 27, 2002)
By Kris Kopicki <krisk4@mac.com>

Provides Mac OS specific features, such as:

  • jEdit exits cleanly when using Command-Q or "Quit" from the "File" menu.
  • Files created or associated with jEdit can now be opened from the Finder, by either double-clicking the file's icon or dragging the file's icon onto the jEdit application icon.
  • jEdit is assigned as the creator of new files.
  • "Preferences..." in the application menu can be used to access jEdits Global Options.
  • "About jEdit..." in the application menu can be used to access the About Dialog. (must be using the new 4.0 application bundle)

Known Issues:

  • Does not work under Mac OS X 10.0.x, as the MRJToolkit routines were not implemented until Mac OS X 10.1.
  • "Preferences..." in the application menu will only work under Mac OS X 10.1.3 or later. If using 10.1.3, the "Java 1.3.1 update 1" must be installed.
  • Files opened in the Finder before jEdit is running are not opened in jEdit.
