It is highly recommended that you install some plugins to make jEdit even more useful to you.

For the latest developments in the world of jEdit, visit www.jedit.org.

If you have upgraded from jEdit 3.2, you should check out what's new in version 4.0.

To create a new file... select File>New.
To edit an existing file... select File>Open.
To customize jEdit... select Utilities>Global Options.
To install plugins... select Plugins>Plugin Manager.
To show this window... select Help>jEdit Help.
If you think you have found a bug, please consult the Frequently Asked Questions document first.
