You can click on various parts of the status bar:
  • Double-clicking the caret position display opens the Go to Line dialog box.
  • Double-clicking the edit mode or encoding indicator opens the Buffer Options dialog box.
  • Clicking the fold mode indicator changes it ("none", to "indent", to "explicit").
  • Clicking the selection status indicator toggles it ("single" or "multi").
  • Clicking the insert mode indicator toggles it ("ins" or "ovr").
  • Double-clicking the memory indicator opens the Memory Status dialog box.
