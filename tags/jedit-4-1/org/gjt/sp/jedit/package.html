This package contains jEdit's core classes.

For plugin development, the most important classes are:

  • EditPlugin
  • View
  • EditPane
  • JEditTextArea
  • Buffer
The EditBus framework is used to receive notification of various editor state changes:
  • EBPlugin
  • EditBus
  • EBComponent
  • EBMessage
For a description of actions.xml file syntax, see the ActionSet class.

For a description of dockables.xml file syntax, see the DockableWindowManager class.
