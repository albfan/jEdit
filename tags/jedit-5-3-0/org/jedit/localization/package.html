This folder contains property files for localization in different languages. To add another localization to jEdit, you can add a new property file to this folder, but be sure to also fix the following properties in jedit_gui.props.

  1. Update the available.lang to list the 2-letter localization code for the language you added.
  2. Add a new property for the "display name" of the new language, preferably in the language itself. For example: 
    options.appearance.lang.ja=日本語
