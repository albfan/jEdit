JEDIT 4.0 VERSION HISTORY

* Version 4.0pre1

+ New Features

- Ant is used instead of jmk to compile jEdit
- JavaHelp library from Sun is now used for the help system. JavaHelp
  supports full-text search, indexing, and printing.

+ Enhancements

- HTML documentation is now generated using DocBook XSL 1.44 stylesheets
