Global Options dialog box

There are many settings that can be changed in this dialog box.
  • To change the text font, go to Text Area.
  • To change the tab width and other editor settings, go to Editor.
  • To dock plugin windows into the view, go to Docking.
  • To change other plugin settings, go to Plugin Options.
  • If you're behind an HTTP proxy and would like to use the plugin manager to install plugins, go to HTTP Proxy.
